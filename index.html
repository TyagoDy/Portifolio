Quem sou eu?

Sou um aspirante a desenvolvedor web com inglês fluente, atualmente estudando Análise e Desenvolvimento de Sistemas (ADS) pela UNA. Acredito que o aprendizado contínuo e a prática hands-on são essenciais para construir soluções web eficazes e inovadoras.

Atualmente curso Análise e Desenvolvimento de Sistemas (ADS) pela UNA, uma formação focada na criação, análise, manutenção e evolução de sistemas de software. O curso está fortalecendo minha base em lógica de programação, algoritmos, estruturação de soluções computacionais e desenvolvimento de aplicações — conhecimentos que complementam diretamente minha jornada como desenvolvedor web em ascensão, com foco em criar interfaces intuitivas, funcionais e bem estruturadas.
Nas horas vagas, dedico-me ao estudo aprofundado de Python e desenvolvimento web moderno, trabalhando em projetos pessoais que variam de pequenas utilidades a aplicações web completas. Essa prática constante me permite aplicar imediatamente os conceitos aprendidos e construir um portfólio sólido.
